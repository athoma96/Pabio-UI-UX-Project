# Pabio – Furniture Rental Web App

A full-stack UI/UX project built for a rent-to-own furniture platform
targeting renters and students who want affordable, stylish living spaces.

## Overview
Pabio was designed and developed as part of CS422: User Interface Design
and Programming at the University of Illinois at Chicago. The project
followed a complete human-centered design process from user research
through high-fidelity prototyping and final UI implementation.

## Features
- Multi-page responsive web app built with React.js, JavaScript, HTML & CSS
- Browse furniture by style themes (Scandinavian, Industrial, Boho, Art Deco)
- Search bar with popular searches and category filtering
- Product pages with similar item recommendations
- Shopping cart with add, remove, and buy-now functionality
- Checkout and payment flow
- Order confirmation with personalized recommendations

## Tech Stack
- **Frontend:** React.js, JavaScript, HTML, CSS
- **Design Pattern:** Model-View-Controller (MVC)
- **Prototyping Tools:** Balsamiq, Figma, Uizard, InVision

## Design Process
1. User research via focus groups (6 participants)
2. Affinity diagramming and data analysis
3. Personas, scenarios, and storyboards
4. Task analysis and wireframe flow diagrams
5. Low-fidelity and high-fidelity prototyping
6. User testing with task completion metrics (100% / 83% / 67% success rates)
7. Iterative design changes based on feedback
